# ログイン機能 テストモデル（シーケンス図）

テストベース：ATRS要件定義書 A03-01（ログインする）・A03-02（ログアウトする）

---

## シーケンス図

```mermaid
sequenceDiagram
    actor ユーザー
    participant ATRSシステム

    ユーザー->>ATRSシステム: ログイン要求（会員番号・パスワード）

    alt 入力情報に誤りあり
        ATRSシステム-->>ユーザー: 入力エラーメッセージ表示
    else 会員番号なし / パスワード誤り
        ATRSシステム-->>ユーザー: 認証失敗メッセージ表示
    else 会員情報の更新に失敗
        ATRSシステム-->>ユーザー: ログイン失敗メッセージ表示
    else 正常
        ATRSシステム-->>ユーザー: ログイン成功（会員名表示）

        ユーザー->>ATRSシステム: ログアウト要求

        alt 会員情報の更新に失敗
            ATRSシステム-->>ユーザー: ログアウト失敗メッセージ表示
        else 正常
            ATRSシステム-->>ユーザー: ログアウト完了
        end
    end
```

---

## カバレッジアイテム（シナリオ）

図を端から端まで辿れるシナリオ数 = **5件**

| # | シナリオ |
|---|---|
| S1 | 入力情報に誤りあり |
| S2 | 会員番号なし / パスワード誤り |
| S3 | ログイン時：会員情報の更新に失敗 |
| S4 | 正常ログイン → ログアウト時：会員情報の更新に失敗 |
| S5 | 正常ログイン → 正常ログアウト |
